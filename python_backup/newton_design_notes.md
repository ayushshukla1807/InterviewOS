
# Newton School NST/Elevate Platform — Design Analysis (From Screenshots)

## Overall Design
- Background: Very dark navy/slate (#0a0e1a or similar)
- Font: Clean sans-serif (likely Inter)
- Two-column layout: Left panel (interview info) + Right panel (instructions)
- Card: Rounded, slightly lighter dark background with subtle border

## Header
- Newton School logo (white text + shield icon) top-left
- "elevate" branding top-right (teal/green accent, wavy logo icon)
- Minimal, clean header, no full nav bar

## Left Panel — Interview Info Card
- Title: "NST Mock Interview I Javascript I ADYPU" (bold, white text)
- Section header: "INTERVIEW TEAM" (uppercase, small gray text, letter-spaced)
- **Invigilator Card**:
  - Avatar: Real photo (male, suit) in a circle
  - Name: "Invigilator" (bold white)
  - Subtitle: "Syed, Newton School" (gray)
  - Yellow dot warning: "Syed will watch your complete interview recording" (orange/amber text)
  - Card has slightly lighter background, subtle border
- **AI Interviewer Card**:
  - Avatar: Female avatar illustration (Ava)
  - Name: "AI Interviewer" (bold white)
  - Subtitle: "Ava" (gray)
  - No warning text
- Section: "INTERVIEW DETAILS" (uppercase, gray)
  - Time icon + "TIME" label + "60 mins" value
  - Document icon + "QUESTIONS" label + "4" value
  - Side by side in a row

## Right Panel — Instructions
- Title: "INSTRUCTIONS" (uppercase, bold, white)
- **Before You Begin** (bold, larger white text)
  - 5 bullet points: quiet environment, camera/mic check, laptop only, stable internet
- **During the Interview** (bold, larger white text)
  - 5 bullet points: fullscreen, no tab switching, responses recorded, code + communication, no external help
- Checkbox: "I agree to Privacy Policy and consent to be interviewed by an AI" (blue link for Privacy Policy)
- Button: "Start Interview" (large, blue/indigo, full width, disabled until checkbox checked)
- Text below: "Please accept the Privacy Policy to start your interview" (gray, small)
- Bottom disclaimer bar: "This interview is being conducted by an AI. It can make mistakes. The final decision is made by a human." + "View Privacy Policy" link (right)

## Feedback/Report Page
- Candidate name, email, phone at top
- "Reveal Score" button
- Interview Details: Start Time, End Time, Duration, Exceeded time
- Communication Skills section with Score badge (3.8/5)
  - Sub-scores: Vocal Delivery 4/5, Expression Clarity 3.9/5, Structured Communication 3.5/5
- Question details section with numbered questions, weightage %, time, difficulty, status, score
- Each question expandable: Question / Candidate's code / Detailed Evaluation tabs

## Color Palette
- Background: #0b0f1a (very dark navy)
- Card: #141928 (dark blue-gray)
- Border: #1e2740 (subtle)
- Accent/Button: #3b5bdb (indigo blue) or #4f46e5
- Warning dot: #f59e0b (amber)
- Elevate brand: teal/green gradient
- Text primary: #ffffff
- Text secondary: #8892a4 (gray)
- Badge green: #22c55e
- Elevate logo: appears to be a stylized wave/tilde in teal
